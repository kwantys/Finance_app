Інформаційне та програмне забезпечення системи управління фінансами

Бакалаврська робота

Опис роботи

Android-додаток, що дозволяє користувачам ефективно управляти персональними фінансами: планування витрат, ведення бюджету, аналітика.

Актуальність теми

У сучасному світі цифрова трансформація охоплює й сферу особистих фінансів. Збільшення обсягу безготівкових операцій вимагає інструментів для зручного контролю доходів і витрат.

Мета та завдання дослідження

Мета: Розробка Android-додатку для управління особистими фінансами.

Основні завдання:

  1. Проаналізувати існуючі мобільні рішення у сфері фінансового обліку.
  2. Сформулювати вимоги до нового програмного забезпечення.
  3. Реалізувати додаток з основним функціоналом.
  4. Забезпечити інтерфейс зручний для користувача.
  5. Оцінити ефективність реалізованої системи.

Методологія дослідження

Під час розробки було використано методи аналізу, проєктування програмного забезпечення, прототипування, тестування та порівняльного оцінювання. Для реалізації застосовано мову Java у середовищі AndroidStudio.

Очікувані результати

Отримання повнофункціонального Android-додатку, що дозволяє користувачам контролювати свої фінанси, візуалізувати витрати та отримувати аналітичні підказки щодо оптимізації бюджету.
